**Goal:** Replace the modal onboarding with a full-page `/start` wizard (申請→審核→判定→發證) that issues a shareable 追星證 with QR loop-closure, rarity badge, and hero+lineup layout, and slim the home page to hero + 圖鑑 with header-nav entries.

**Architecture:** A localStorage-backed wizard state (`kstar:wizard`) drives four URL-tracked steps under `/start`; existing engines (`questionnaire.ts`, `archetypes.ts`, `DevelopFourCuts`, `exportImage.ts`) are reused, with quiz screen-building extracted to a shared lib so the existing modal keeps working. The card is a new `FanIdCard` component; rarity comes from a generated `typeRarity.ts` (same pattern as `tokenStats.ts`); the QR is a build-time static SVG.

**Tech Stack:** Next.js 16 App Router (async `params`/`searchParams` are Promises; client components using `useSearchParams` need a Suspense boundary), React 19, Tailwind v4 (CSS-first in `globals.css` `@theme`), html-to-image (hardened recipe in `exportImage.ts`), `qrcode` (devDependency, build-time only).

## Global Constraints

- **Verification gates (this repo has no unit-test framework):** `npm run build` is the typecheck gate; logic modules get `npx tsx` verify scripts under `scripts/`; UI gets preview walkthroughs on the `kpop-dev` launch config (port 3100). Do not introduce a test framework in this slice.
- **Copy blocklist (hard):** never 拉踩/控評/反黑/塌房/泥塑/打榜/內卷/隊推/團推/站姐/後援會入會/官方認證. Card = **追星證** (en FAN ID); idol photocards = **小卡**; four-cut = **人生四格**.
- **No fake scarcity:** no 期數, no membership tiers. Serial + 發證日期 are fine.
- **Catalog never reaches the client.** Client components receive `ArtistLite`/`CardArtist` via props (`src/lib/lite.ts`). Any new server data goes through `src/lib/data.ts`.
- **SSR guard pattern:** localStorage readers use lazy `useState` initializers with `typeof window` checks, or mount-gated `useEffect` (see `SoulPortraitButton.tsx`).
- **Flash consent:** tentpole shutter flashes play ONLY if the user accepted the flash intro (Direction D). Persist that choice; default to the soft no-flash develop otherwise.
- **Export recipe is law:** any new export target uses `exportNode()` from `src/lib/exportImage.ts` unchanged (skipFonts, style-prop allowlist, decode-wait, 3× retry, PAD margin).
- **prefers-reduced-motion is intentionally not honored** (product decision, `globals.css` §3) — do not add media-query gates; the consent dialog is the accessibility mechanism.
- Commit after each task (local only, no push). Conventional prefixes as in `git log` (`feat(…)`, `fix(…)`, zh scope names welcome).

```ts
// Resumable state for the /start wizard. One JSON blob under kstar:wizard.
// All readers parse leniently (old/partial blobs must never throw) — same
// discipline as UserPrefs. finishWizard() is the ONLY writer of kstar:prefs.
import type { ScoreLayer, UserPrefs } from "./types";
import { SCORE_LAYERS } from "./types";
import { rankToWeights } from "./questionnaire";

export interface WizardState {
  step: 0 | 1 | 2 | 3 | 4;
  picks: string[];            // idol ids, 0..4, user-ordered (slot order)
  rank: ScoreLayer[];         // 入坑優先序
  answers: Record<string, string>;
  archetype?: string;         // canonical mixed-case code once computed
  heroId?: string;            // spotlight 本命 (defaults to picks[0])
  fanName?: string;           // optional 持卡人
  song?: { title: string; artist: string; artworkUrl: string } | null;
}

const KEY = "kstar:wizard";

const EMPTY: WizardState = { step: 0, picks: [], rank: [...SCORE_LAYERS], answers: {} };

export function loadWizard(): WizardState {
  if (typeof window === "undefined") return { ...EMPTY };
  try {
    const raw = localStorage.getItem(KEY);
    if (!raw) return { ...EMPTY };
    const p = JSON.parse(raw) as Partial<WizardState>;
    return {
      step: ([0, 1, 2, 3, 4] as const).includes(p.step as 0) ? (p.step as WizardState["step"]) : 0,
      picks: Array.isArray(p.picks) ? p.picks.filter((x) => typeof x === "string").slice(0, 4) : [],
      rank: Array.isArray(p.rank) && p.rank.length === 4 ? (p.rank as ScoreLayer[]) : [...SCORE_LAYERS],
      answers: p.answers && typeof p.answers === "object" ? (p.answers as Record<string, string>) : {},
      archetype: typeof p.archetype === "string" ? p.archetype : undefined,
      heroId: typeof p.heroId === "string" ? p.heroId : undefined,
      fanName: typeof p.fanName === "string" ? p.fanName : undefined,
      song: p.song && typeof p.song === "object" ? (p.song as WizardState["song"]) : null,
    };
  } catch {
    return { ...EMPTY };
  }
}

export function saveWizard(patch: Partial<WizardState>): WizardState {
  const next = { ...loadWizard(), ...patch };
  try { localStorage.setItem(KEY, JSON.stringify(next)); } catch { /* quota — state stays in memory */ }
  return next;
}

export function clearWizard(): void {
  try { localStorage.removeItem(KEY); } catch { /* ignore */ }
}

// Merge the finished wizard into the app's existing prefs shape and mark done.
export function finishWizard(s: WizardState): void {
  let prefs: Partial<UserPrefs> & { fanIdClaimed?: boolean; fanName?: string; heroId?: string } = {};
  try { prefs = JSON.parse(localStorage.getItem("kstar:prefs") ?? "{}"); } catch { /* fresh */ }
  const merged = {
    ...prefs,
    topIdols: s.picks,
    weights: rankToWeights(s.rank),
    layerRank: s.rank,
    archetype: s.archetype,
    heroId: s.heroId ?? s.picks[0],
    fanName: s.fanName,
    fanIdClaimed: true,
  };
  localStorage.setItem("kstar:prefs", JSON.stringify(merged));
  localStorage.setItem("kstar:onboarding", "done");
  window.dispatchEvent(new Event("kstar:prefs-updated"));
  clearWizard();
}
```

// Always-flattering rarity framing (owner-ratified): rare types flex the
// number; common types get a badge, never a demotion.
import { TYPE_RARITY } from "./typeRarity";

export function frameRarity(code: string): { label: string; pct: number } {
  const pct = TYPE_RARITY[code] ?? 6.3; // fallback ≈ uniform
  if (pct <= 4) return { label: `${pct}% 超稀有型別`, pct };
  if (pct <= 8) return { label: `${pct}% 稀有型別`, pct };
  if (pct <= 14) return { label: `${pct}% 同路人`, pct };
  return { label: "主流大勢型", pct };
}

"use client";

// Persistent wizard shell: logo, 1–4 stepper with checkmarks, fixed Back/Next.
// Pure presentation — step logic lives in the page.
import Link from "next/link";
import { copy } from "@/lib/copy";

const STEPS = [copy.wizStep1, copy.wizStep2, copy.wizStep3, copy.wizStep4];

export default function WizardChrome({
  step, canNext, onBack, onNext, nextLabel, children,
}: {
  step: 1 | 2 | 3 | 4;
  canNext: boolean;
  onBack?: () => void;
  onNext?: () => void;
  nextLabel?: string;
  children: React.ReactNode;
}) {
  return (
    <div className="mx-auto flex min-h-[calc(100vh-3.5rem)] w-full max-w-2xl flex-col px-4 py-6">
      <Link href="/" className="mb-4 self-center font-orbitron text-lg font-black chrome-text">
        {copy.appName}
      </Link>
      <ol className="mb-6 flex items-center gap-2" aria-label="步驟">
        {STEPS.map((label, i) => {
          const n = (i + 1) as 1 | 2 | 3 | 4;
          const done = n < step;
          const on = n === step;
          return (
            <li key={label} className="flex flex-1 items-center gap-2 last:flex-none">
              <span
                className={`grid h-7 w-7 shrink-0 place-items-center rounded-full border text-xs font-bold ${
                  done ? "border-emerald-500 text-emerald-600" :
                  on ? "border-[#b4302b] bg-[#b4302b] text-white" :
                  "border-[#c8ccd2] text-[#9aa0aa]"
                }`}
                aria-current={on ? "step" : undefined}
                title={label}
              >
                {done ? "✓" : n}
              </span>
              {i < STEPS.length - 1 && <span className="h-px flex-1 bg-[#c8ccd2]/50" />}
            </li>
          );
        })}
      </ol>
      <div className="flex-1">{children}</div>
      <div className="mt-6 flex gap-3">
        {onBack && (
          <button onClick={onBack} className="flex-1 rounded-xl border border-[#c8ccd2] py-3 font-bold">
            {copy.wizBack}
          </button>
        )}
        {onNext && (
          <button
            onClick={onNext}
            disabled={!canNext}
            className="flex-1 rounded-xl bg-[#1c1e24] py-3 font-bold text-white disabled:opacity-40"
          >
            {nextLabel ?? `${copy.wizNext} (${step}/4)`}
          </button>
        )}
      </div>
    </div>

// /start orchestrator. step 0 = landing poster; 1–4 = wizard. URL ?step= is

import { useEffect, useState } from "react";
import Link from "next/link";
import { useRouter, useSearchParams } from "next/navigation";
import type { ArtistLite } from "@/lib/lite";
import { copy } from "@/lib/copy";
import { loadWizard, saveWizard, type WizardState } from "@/lib/wizardState";
import WizardChrome from "@/components/wizard/WizardChrome";

export default function StartFlow({ allArtists }: { allArtists: ArtistLite[] }) {
  const router = useRouter();
  const params = useSearchParams();
  const [wiz, setWiz] = useState<WizardState | null>(null); // null until mounted (SSR guard)

  useEffect(() => {
    let s = loadWizard();
    if (params.get("claim") === "1") {
      try {
        const prefs = JSON.parse(localStorage.getItem("kstar:prefs") ?? "{}");
        if (Array.isArray(prefs.topIdols) && prefs.topIdols.length === 4) {
          s = saveWizard({
            picks: prefs.topIdols,
            rank: prefs.layerRank ?? s.rank,
            archetype: typeof prefs.archetype === "string" ? prefs.archetype : undefined,
            step: typeof prefs.archetype === "string" ? 4 : 2,
          });
        }
      } catch { /* fall through to normal flow */ }
    }
    const urlStep = Number(params.get("step"));
    if ([1, 2, 3, 4].includes(urlStep) && urlStep <= s.step) s = { ...s, step: urlStep as WizardState["step"] };
    setWiz(s);
  }, [params]);

  if (!wiz) return null;

  const go = (step: WizardState["step"]) => {
    const s = saveWizard({ step });
    setWiz(s);
    router.replace(step === 0 ? "/start" : `/start?step=${step}`);
  };

  if (wiz.step === 0) {
    return (
      <div className="mx-auto flex min-h-[calc(100vh-6rem)] max-w-xl flex-col items-center justify-center gap-6 px-4 text-center">
        <span className="font-orbitron text-2xl font-black chrome-text">{copy.appName} 發證中心</span>

          href="/#idols"

          className="text-sm text-[#9aa0aa] hover:text-[#5e636d]"
        >
          {copy.wizEscape}
        </Link>
      </div>
    );
  }

  // Steps 1–4 placeholders — replaced by StepPicker/StepQuiz/StepReveal/StepIssue in Tasks 6/7/8/9.
  return (
    <WizardChrome
      step={wiz.step as 1 | 2 | 3 | 4}
      canNext
      onBack={wiz.step > 1 ? () => go((wiz.step - 1) as WizardState["step"]) : undefined}

// Step 1 建立本命欄: binder (4 equal slots, tap-tap to swap order) on top;
// search + filterable photo grid below (picker-skinned 圖鑑 — photo+name only).
// Each insert soft-develops (.wiz-develop). 4th insert → tentpole flash

import { useMemo, useState } from "react";
import type { ArtistLite } from "@/lib/lite";
import { GENDER_OPTIONS, GEN_OPTIONS, matchesLite, type GenderFilter, type GenFilter } from "@/lib/browse";
import Thumb from "@/components/Thumb";
import { copy } from "@/lib/copy";

const PAGE = 24;

export default function StepPicker({
  allArtists, picks, onChange,
}: {
  allArtists: ArtistLite[];
  picks: string[];
  onChange: (picks: string[]) => void;
}) {
  const [q, setQ] = useState("");
  const [gender, setGender] = useState<GenderFilter>("全部");
  const [gen, setGen] = useState<GenFilter>("全部");
  const [page, setPage] = useState(0);
  const [swapFrom, setSwapFrom] = useState<number | null>(null); // tap-tap rearrange
  const [flash, setFlash] = useState(false);

  const byId = useMemo(() => new Map(allArtists.map((a) => [a.id, a])), [allArtists]);
  const ql = q.trim().toLowerCase();
  const filtered = useMemo(
    () =>
      allArtists.filter(
        (a) =>
          matchesLite(a, { gender, gen, pos: "全部" }) &&
          (!ql || a.name.toLowerCase().includes(ql) || (a.name_zh ?? "").toLowerCase().includes(ql)),
      ),
    [allArtists, gender, gen, ql],
  );
  const pageItems = filtered.slice(page * PAGE, (page + 1) * PAGE);

  function toggle(id: string) {
    if (picks.includes(id)) return onChange(picks.filter((p) => p !== id));
    if (picks.length >= 4) return;
    const next = [...picks, id];
    onChange(next);
    if (next.length === 4 && localStorage.getItem("kstar:flashOk") === "1") {
      setFlash(true);
      setTimeout(() => setFlash(false), 750);
    }

    if (swapFrom === null) return setSwapFrom(i);
    const next = [...picks];
    [next[swapFrom], next[i]] = [next[i], next[swapFrom]];
    setSwapFrom(null);
    onChange(next);

      {/* binder — 4 EQUAL slots (owner-ratified: no visual hierarchy) */}

            <button
              key={a.id}
              onClick={() => slotTap(i)}
              className={`wiz-develop relative aspect-[3/4] overflow-hidden rounded-lg border ${
                swapFrom === i ? "border-[#b4302b] ring-2 ring-[#b4302b]" : "border-[#c8ccd2]"
              }`}
              title={`${a.name} — 點兩格交換位置`}
            >
              <Thumb artist={a} className="h-full w-full object-cover" />
              <span
                onClick={(e) => { e.stopPropagation(); toggle(a.id); }}
                className="absolute right-1 top-1 grid h-5 w-5 place-items-center rounded-full bg-black/60 text-xs text-white"
                role="button" aria-label={`移除 ${a.name}`}
              >×</span>
            </button>
          ) : (
            <div key={i} className="grid aspect-[3/4] place-items-center rounded-lg border border-dashed border-[#c8ccd2] text-[#c8ccd2]">＋</div>
          );
        })}
      </div>
      <p className="mb-3 text-center text-xs text-[#9aa0aa]">{copy.wizPickHint(picks.length)}</p>

      <input
        value={q}
        onChange={(e) => { setQ(e.target.value); setPage(0); }}
        placeholder={copy.searchPlaceholder}
        className="mb-3 w-full rounded-xl border border-[#c8ccd2] px-4 py-2.5"

            className={`rounded-full border px-3 py-1 text-xs ${gender === g ? "border-[#b4302b] bg-[#b4302b]/10 text-[#b4302b]" : "border-[#c8ccd2] text-[#5e636d]"}`}>
            {g}
          </button>
        ))}
        {GEN_OPTIONS.map((g) => (
          <button key={g} onClick={() => { setGen(g); setPage(0); }}
            className={`rounded-full border px-3 py-1 text-xs ${gen === g ? "border-[#b4302b] bg-[#b4302b]/10 text-[#b4302b]" : "border-[#c8ccd2] text-[#5e636d]"}`}>

          </button>
        ))}
      </div>

      <div className="grid grid-cols-4 gap-2 sm:grid-cols-6">
        {pageItems.map((a) => {
          const picked = picks.includes(a.id);
          return (
            <button key={a.id} onClick={() => toggle(a.id)}
              className={`relative aspect-[3/4] overflow-hidden rounded-lg border text-left ${picked ? "border-[#b4302b] ring-2 ring-[#b4302b]" : "border-[#c8ccd2]/60"}`}
              aria-pressed={picked}>

        })}
      </div>

        <div className="mt-3 flex justify-center gap-2 text-sm">

        </div>

    </div>
  );
